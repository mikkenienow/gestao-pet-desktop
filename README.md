# Gestão Pet
O Gestão Pet é um projeto desenvolvido em Java para uso local, que possui um banco de dados em SQLite. O sistema tem como objetivo facilitar a gestão de estabelecimentos relacionados a cuidados de animais de estimação, como pet shops, clínicas veterinárias, entre outros. Ele permite o cadastro e controle de Pets, clientes, fornecedores e funcionários, além de oferecer funcionalidades de controle de estoque e controle financeiro.

> ### Hospedagem diferenciada
> A funcionalidade de hospedagem possui um diferencial interessante. Caso seja necessário reservar vários dias consecutivos para a hospedagem de um Pet, o Gestão Pet verifica automaticamente a disponibilidade de alojamentos livres pelo total de dias solicitados. Em caso de falta de um alojamento único para o período desejado, o sistema automaticamente divide a hospedagem em vários alojamentos, garantindo acomodações adequadas para o Pet. </br>
